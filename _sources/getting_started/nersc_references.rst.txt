General NERSC References
=========================-

- `NERSC User Documentation <https://docs.nersc.gov>`_
- NERSC provides a number of training events; many are recorded and available
  for later viewing.
- `An overview of Perlmutter <https://docs.nersc.gov/systems/perlmutter/>`_,
  NERSC's current system.
- An overview of the
  `SLURM workload manager <https://docs.nersc.gov/jobs/scheduler/>`_,
  which manages the batch system at NERSC, along with
  `example batch scripts <https://docs.nersc.gov/jobs/examples/>`_
  for submitting jobs on Perlmutter.
- The `queue wait time page <https://my.nersc.gov>`_ is a useful resource for
  understanding queue wait times and configuring your batch job to start faster.
  *(NERSC login required)*
